# Zernio Plugin for Cursor

The Zernio plugin for [Cursor](https://cursor.com) connects Cursor to the Zernio Social Media Scheduling API via MCP.

## What's Included

- **MCP Server** - Full access to the [Zernio API](https://docs.zernio.com) for scheduling posts, managing accounts, uploading media, running ads, managing contacts, and more across 14 social platforms

## Prerequisites

- A [Zernio API key](https://zernio.com) (free tier available)
- Set `ZERNIO_API_KEY` in your environment
- Python 3.10+ (for the MCP server via `uvx`)
